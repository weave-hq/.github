# Weave HQ

We build systems that amplify human capability through coordinated intelligence.

## Loom

Loom is a platform for coordinated intelligence.

It orchestrates specialized programs to investigate, reason, and act across systems—scaling what operators can do without replacing them.

## How Loom works

Loom is built around a small set of core concepts:

- **Loom Control** — orchestration and execution lifecycle
- **Programs** — specialized units of capability
- **Cycles** — executions of coordinated work
- **Integrations** — connected systems and tools

## Why Loom

We believe systems like Loom do not replace human operators.

Like the industrial loom increased the demand for skilled textile work, Loom increases the scale and effectiveness of human capability through coordination.

## Repositories

- `loom-control` — orchestration and execution

- `loom-ui` — user interface
